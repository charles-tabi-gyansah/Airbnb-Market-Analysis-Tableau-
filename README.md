# Airbnb Market Analysis with Tableau

## Project Overview
This project presents an end-to-end Airbnb market analysis using Tableau. The goal was to explore pricing trends, host activity, and demand patterns through interactive dashboards that support data-driven decision-making.
The analysis focuses on transforming cleaned Airbnb data into meaningful visual insights for business and market evaluation.



## Tools & Technologies
- Tableau Desktop
- Interactive Dashboards & Visual Analytics
- Calculated Fields and Filters
- Data Visualization Best Practices


## Dataset Description
The dataset includes Airbnb listing information such as:
- Listing price
- Neighborhood and location
- Host activity and number of listings
- Availability and review metrics
- Room type and property details

## Analysis Performed
- Pricing trends across neighborhoods
- Identification of the most active hosts
- Distribution of listings by room type
- Review volume as an indicator of demand
- Availability patterns across locations


## Dashboards & Visualizations
- Pricing comparison by neighborhood
- Host activity and listing concentration
- Demand insights using review counts
- Interactive filters for neighborhood, room type, and price range


## Key Skills Demonstrated
- Data storytelling with Tableau
- Dashboard design and layout
- Business-focused visual analysis
- Use of filters, parameters, and calculated fields
- Translating data into actionable insights
